The Process

01

Research & Wireframes

I started by mapping out every page in Figma. Low-fidelity wireframes for the homepage, product page, detail page and cart. No colors yet — just structure and flow.

02

UI Design in Figma

Once the wireframes were done I moved to Figma. Color palette, typography, buttons, cards — all documented in a style guide before writing a single line of code.

03

Built it

Translated the Figma design into a working website using HTML, CSS and JavaScript. Every page from the wireframe got built — homepage, product page, detail page and cart.
